# 🥷 Personal Portfolio Website

A modern, responsive personal portfolio built with React to showcase my projects, experience, and learning journey.

---

## ✨ Features

- 🧩 **Project Showcase**
  - Interactive project cards
  - Flip-to-preview image carousel
  - Dedicated project pages rendered from Markdown

- 🔗 **Dynamic GitHub Integration**
  - Automatically fetches latest commit dates from GitHub
  - Fallback system for API limits

- 🎨 **Light & Dark Mode**
  - Persistent theme using localStorage
  - Clean and consistent UI across themes

- 📂 **Project Detail Pages**
  - Markdown-driven content
  - Reusable layout for all projects
  - Direct links to source code

- 🧠 **Notes Section (WIP)**
  - Structured learning notes by topic
  - Individual note pages

- 🏆 **Certifications Section**
  - Grid-based layout
  - Status tracking (Planned, In Progress, Completed)

- 📱 **Responsive Design**
  - Works across desktop and mobile
  - Optimised layouts for smaller screens

---

## 🛠️ Tech Stack

- **React** (Vite)
- **React Router**
- **Tailwind CSS**
- **GitHub REST API**
- **Markdown Rendering**

---

## 📚 What I Learned

This project helped me develop and improve:

- Building scalable React component structures
- Client-side routing with React Router
- Handling API requests and rate limits
- Designing reusable UI components
- Managing theme state and persistence
- Working with Markdown content dynamically
