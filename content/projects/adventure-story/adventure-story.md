# 🌋 Adventure Story

A React-based choice-driven adventure game where players explore different paths, encounter enemies, and progress through a branching storyline.

---

## ✨ Features

- 🧭 **Choice-based gameplay**
  Progress through the story by selecting different paths and decisions

- ⚔️ **Basic combat system**
  Engage in simple encounters with enemies

- 🗺️ **Multiple routes & outcomes**
  Different choices lead to different scenarios and progression

- 🔑 **Inventory / progression elements**
  Collect items such as keys to unlock new paths

- 🎮 **State-driven UI**
  Game flow handled through React state updates

---

## 🛠️ Tech Stack

- **React** (Vite)
- **Tailwind CSS**
- **JavaScript**

---

## 📚 What I Learned

This project helped me explore:

- Managing game state in React
- Structuring logic for branching story paths
- Handling user input and conditional rendering
- Designing simple game mechanics (combat, progression)
- Breaking down a larger idea into smaller components
- Understanding when a project becomes too complex and needs rethinking

---

## ⚠️ Project Status

**Abandoned / On Hold**

This project was paused as the scope began to grow significantly.
While the core idea works, the structure would need a rethink to properly scale into a full game.

---

## 💡 Future Improvements (if revisited)

- Refactor state management (possibly using a more structured approach)
- Improve story organisation (data-driven story system)
- Expand combat mechanics
- Add animations and better visual feedback
- Save/load game progress
